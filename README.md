# ITStep_JS_Apollo_Shop

* :ambulance:Execute This commands
```bat
cd client
npm i @nuxtjs/apollo
npm i @nuxtjs/axios
npm i core-js
npm i graphql-tag
npm i nuxt

cd ../server
npm i @prisma/client
npm i apollo-server-constraint-directive
npm i apollo-server-express
npm i cors
npm i express
```

* To start the project
1. Execute this on new console
```bat
cd server
node app.js
```
1. Execute this on new console
```bat
cd client
npm run dev
```
